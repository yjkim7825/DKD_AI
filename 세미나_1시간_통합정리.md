# DKD 다중오믹스 논문 — 1시간 세미나 통합 정리

**논문**: Multi-omics and machine learning identify FN1 and ALDH2 as stage-specific biomarkers in diabetic kidney disease (J. Lin et al., *Renal Failure*)

**핵심 한 줄**
> DKD에서 **FN1(위험·섬유화 축)·ALDH2(보호·산화스트레스 축)** 를 5가지 방법으로 교차검증해 **단계특이 바이오마커**로 확립하고, **레스베라트롤**을 이중표적 치료 후보로 제시.

**발표 흐름 (이 순서로 말하면 됨)**
```
배경 → [DEG] 뭐가 변하나 → [경로] 무슨 기전 → [MR] 원인인가
→ [교집합] 10개 → [ML] 진단되나 → [임상] 신기능 연결
→ [도킹] 약 있나 → [scRNA] 어느 세포·언제 → 결론(FN1·ALDH2로 수렴)
```

---

## 0. 배경 — 왜 이 연구를 했나

- **DKD** = 당뇨병성 신장질환. 당뇨 환자의 30~40%에서 발생, **말기신부전(ESKD)로 가는 최대 경로**.
- **문제**: SGLT2 억제제 등 좋은 약이 있어도 **DKD를 못 되돌림** → 근본 기전을 아직 모름.
- **한계**: 겉 신호(단백뇨·eGFR)는 이미 신장이 상한 뒤에 나타남 → 그 전에 잡을 **분자 바이오마커**가 필요.
- **공백 2개** (이 논문이 채우는 것):
  1. 진단 + **치료** 겸용 바이오마커 부재 → FN1·ALDH2 + 레스베라트롤로 해결
  2. **초기 vs 말기** 분자 차이 미규명 → 정상·초기·말기 3그룹 + single-cell 단계별 분석으로 해결
- **전략**: bulk RNA-seq + single-cell + MR + ML + 도킹을 **통합**(multi-omics) = 한 층위만 보면 놓치는 걸 교차검증.

```
[GEO bulk RNA-seq]              [eQTLGen + DKD GWAS]
정상·초기·말기 발현행렬            SNP→발현 / SNP→DKD
      │ DEG                          │ MR
      ▼                              ▼
 다르게 변한 유전자             인과 유전자 366개
 (+ 경로분석: 무슨 기전)         (위험/보호)
      └───────┬──────────────────────┘
              ▼ 교집합 (발현변화 + 인과)
          후보 10개
              │ ML (LASSO → 4모델, 진단력)

```

**단계별 "입력 → 처리 → 출력" (표로 한 번에)**

| 단계 | 입력 데이터 | 처리 | 출력 |
| --- | --- | --- | --- |
| 1. DEG | 발현행렬(정상/초기/말기) | 그룹 간 차이 (limma, |logFC|>0.585, FDR<0.05) | 다르게 발현된 유전자 |
| 2. 경로 | DEG + 경로별 유전자셋(Table S2) | ORA/GSEA/ssGSEA | 활성 병리경로(염증·EMT 등) |
| 3. MR | eQTL(노출) + DKD GWAS(결과) | 발현→DKD 인과 (IVW p<0.05) | 인과 유전자 366개(위험/보호) |
| 4. 교집합 | DEG ∩ MR | 발현변화 + 인과 + 방향일치 | 후보 10개 |
| 5. ML | 후보 10개 + 학습/검증셋 | LASSO 선택 → 4모델 검증 | FN1·ALDH2 |
| 6. 임상 | Nephroseq 발현·eGFR | 상관분석(r) | FN1↓·ALDH2↑ 신기능 연결 |
| 7. 도킹 | FN1·ALDH2 구조 + 화합물(DSigDB) | 결합 시뮬레이션(Vina) | 레스베라트롤(이중표적) |
| 8. scRNA | single-cell(신장·소변) | 세포분류 + pseudotime + 경로 | FN1↑(MES)·ALDH2↓(PCT), 초기·말기 차이 |

**왜 이 순서인가 (발표 한마디)**
> DEG로 **넓게** 후보를 잡고(상관) → MR로 **원인**만 거르고 → 교집합·ML로 **핵심 2개**로 좁힌 뒤 → 임상·도킹·scRNA로 **검증·치료·기전**까지 확장. 즉 **넓게 → 좁게 → 깊게**의 깔때기 구조.

## A. DEG (차등발현 분석) — 뭐가 다르게 발현되나

**왜**: 2만+ 유전자 중 정상·초기·말기에서 **다른 것만** 골라 탐색 범위 축소 (1차 필터). 단, DEG는 **상관**까지만(원인 아님).

**과정 (식·정의)**
- 입력: 발현행렬(정상/초기/말기 샘플 × 유전자), 도구: **limma**
- 기준: **|logFC| > 0.585** (=1.5배) **AND FDR < 0.05**
- **logFC** = log₂(실험군÷대조군). log₂ 씌우는 이유 = 기준점 1→0으로 옮기고 증감을 **대칭**으로 (2배↑=+1, 반토막=−1)
- **FDR**(거짓발견율) = 유의하다고 뽑은 것 중 가짜 비율. 수만 개 동시검정이라 그냥 p<0.05면 가짜 대량 → **Benjamini-Hochberg**로 FDR<0.05 제어 (뽑은 것 중 가짜 5% 미만)
- 출력: DEG 목록 (볼케이노·히트맵으로 표시)

**결과 (논문)**: 진행할수록 DEG 급증

**논문 vs 재현 (DEG 개수)**

| 비교 | 논문 | 우리 재현 |
| --- | --- | --- |
| 말기 vs 초기 | 2,833 (↑1,557/↓1,276) | 3,314 (↑1,755/↓1,559) |
| 말기 vs 정상 | 3,525 | 4,022 |
| 초기 vs 정상 | 390 | 671 |
| 패턴 | 진행할수록 급증 | **동일 ✓** |

> 재현 수치가 조금 높은 건 threshold·전처리 차이. **패턴은 동일**.

## B. 경로 분석 — 무슨 병리기전이 켜졌나

**왜**: 유전자 이름만으론 의미 없음 → **경로(기능) 단위**로 묶어 "무슨 병리과정이 켜졌나" 해석.

**과정 (정의 — 3가지 구분이 중요)**
- **ORA**(과대표현): 내 DEG 목록이 특정 경로에 몰렸나 → **초기하분포(hypergeometric) 검정**, x축=Count, 색=p.adjust (논문이 쓴 방식)
- **GSEA**: 전체 유전자를 순위 매겨 경로가 위/아래 쏠렸나 → NES(정규화 강화점수)
- **ssGSEA**: **샘플/세포마다** 경로 활성 점수 (GSVA/AddModuleScore)
- 유전자셋: **Hallmark 50개**, **KEGG 186개** (Table S2 병리경로 6종: 산화스트레스·염증·EMT·노화·자멸사·자가포식)

**결과 (논문)**: 말기일수록 **EMT·염증·면역** 경로 상위 활성. 검증셋(GSE30529)서도 재현.

**논문 vs 재현**: 말기 상위 경로 **EMT·염증 일치 ✓**. (재현 초기 버전은 GSEA로 그렸다가 논문처럼 **ORA로 재작업**해 맞춤)

## C. MR (멘델리안 무작위화) — 진짜 원인인가 ⭐ (세미나 핵심)

**왜**: DEG는 상관까지만 — "병 때문에 변한 건지(결과), 유전자가 병을 일으킨 건지(원인)" 구분 불가. **치료 표적은 원인이어야** 함. 사람한테 유전자를 실험으로 못 바꾸니, **태어날 때 무작위로 정해진 SNP을 지렛대**로 "발현→DKD" 인과 추정 (자연이 만든 무작위 실험).

**핵심 개념/정의**
- **SNP**: 유전변이. 태생적·평생 불변 → 역인과·교란 차단
- **도구변수(instrument)**: 발현을 대리하는 SNP
- **two-sample MR**: 노출(SNP→발현)과 결과(SNP→DKD)를 **다른 두 데이터**에서 가져와 결합
- **Zscore / beta / SE**: Zscore = beta/SE (연관 강도·방향). beta = 효과크기(회귀계수). SE = 오차막대

**과정 (① eQ 불러오기 → ⑦ IVW → 재현)**

**① 노출 만들기 (eQTLGen)**
- eQTLGen = 31,684명 혈액 eQTL. **Zscore만** 줌 → beta로 변환 필요
- **SNP 선별** (원본 값으로, 변환 전):
  - **Pvalue < 5e-8** (우연일 확률 2천만분의 1 미만) = 발현과 강한 것만 (약한 도구 방지)
  - **LD 클럼핑 r² < 0.001** (kb=10000, EUR 참조패널) = 독립 SNP만, lead SNP 대표
  - → FN1 도구변수 **3개** (rs10932612·rs17525860·rs615857)
- **빈도(2-4, eaf)를 붙여 Zscore→beta/SE 변환**:
  ```
  beta = Z / √(2·p·(1−p)·(N+Z²))     ( p = 대립유전자 빈도 )
  SE   = 1 / √(2·p·(1−p)·(N+Z²))
  ```

**② 결과 데이터 (DKD GWAS)**
- FinnGen / GCST = DKD 환자 vs 대조군을 SNP마다 **로지스틱 회귀** (0/1 병 데이터라 로지스틱)
- 회귀가 **beta_DKD·SE를 원본에 이미** 산출 (변환 불필요). SNP 안 거르고 **그 3개만 조회**

**③ 붙이기 (harmonise)**: 같은 **rsID**로 매칭 + 대립유전자 **방향 정렬**

**④ 기울기 = 인과 크기**
```
기울기 = beta_DKD ÷ beta_발현 = (SNP→DKD)/(SNP→발현) = 발현→DKD   ← SNP 약분!
```

**⑤ IVW로 합치기** (SNP 3개 기울기가 다르니)
- 가중치 = 1/SE² (정확한 SNP에 힘↑) → 합친 beta → **OR = exp(beta)**, p값
- **OR**: >1 위험 / <1 보호 / =1 무영향. **p<0.05** = 유의
- 검증: MR-Egger·Weighted median (같은 데이터 다르게 합쳐 확인)
- (SVM은 MR 아님 — ML 단계)

**⑥ 두 결과 데이터로 재현** (우연 아닌지 교차확인)

**⑦ 방향충돌 제거 → 366개 → DEG 교집합 → 10개**
- FinnGen 214개 ∪ GCST 161개, 방향충돌 4개 제외 = **366개** (Figure 3A 벤: 205+9+152)
- ※ **3개 = FN1 한 유전자의 SNP 수 / 214개 = 유의한 "유전자" 수** (층이 다름! 실제론 5,430개 유전자에 반복)

**결과 (논문)**: 핵심 10개 = 위험 5(CREB5·**FN1**·IFI44L·SYTL2·XAF1) + 보호 5(**ALDH2**·CA1·CDKN1B·TSPYL5·VNN2). 민감도(funnel·leave-one-out·이질성·다면발현·역인과) 다 통과.

**논문 vs 재현 (eQTLGen 원본으로 직접 재현)**

| 유전자 | 논문 Supp9 | 우리(eQTLGen→GCST) | 판정 |
| --- | --- | --- | --- |
| FN1 | OR 2.78, p=0.012 (3 IV) | OR 4.53, p=0.012 (3 IV) | 방향·유의 **일치** ✓ (OR 크기차=노출 Z스케일) |
| ALDH2 | OR 0.673, p=0.032 (14 IV) | OR 0.57, p=0.097 (3 IV) | 방향 **일치**, 유의성은 IV 수(3 vs 14) 한계 |
| FinnGen(둘 다) | null | null (FN1 p=0.79) | **일치** ✓ |

> 핵심 서사(**FinnGen null → GCST 유의, FN1=위험·ALDH2=보호**)를 eQTLGen 원본에서 재현. FN1은 유의성까지 정확 일치, ALDH2는 방향 일치(유의성은 엄격 클럼핑에 따른 IV 수 차이).

## D. 교집합 → 후보 10개

**왜**: "원인(MR) + 발현변화(DEG)" **둘 다 + 방향까지** 맞는 것만 = 신뢰도↑

- Figure 3B = MR 위험(OR>1) ∩ DEG 고발현 → 5개 (벤: 257·5·157)
- Figure 3C = MR 보호(OR<1) ∩ DEG 저발현 → 5개 (벤: 341·5·195)
- → 5+5 = **핵심 10개**

## E. ML (머신러닝 진단모델)

**왜**: 후보 10개도 많음 → 실제 진단에 쓸 것만 추리고 "환자를 정확히 구분하나" 성능 증명.

**과정 (정의)**
1. **배치보정 + PCA** (Figure 4A): 데이터셋별 기술차(배치효과) 제거, PCA로 잘 섞였나 확인. 훈련=GSE96804 / 검증=GSE104948·104954 (과적합 방지)
2. **LASSO** (Figure 4B): 불필요 유전자 계수를 **0으로 눌러** 핵심만 선택. **λ**=규제강도, **10-fold CV**로 최적 λ. → **10개 → 6개**
3. **ROC/AUC** (Figure 4D–G): AUC = ROC 아래 면적, **>0.8=우수**. 검증셋 AUC>0.8만 통과 → **6개 → FN1·ALDH2 2개** (나머지 4개 <0.7 탈락, Figure S4)
4. **발현 방향** (Figure 4C): **FN1↑(위험)·ALDH2↓(보호)** → MR·DEG·ML **세 방법 일치**
5. **조합 성능** (Figure 4H–I): RF·SVM·GLM·XGBoost 4모델, 모두 AUC>0.8. **GLM 최고** (훈련 0.978/검증 0.942). calibration(S5A)·DCA(S5B)로 임상유용성 확인

**결과 (논문)**: 최종 바이오마커 = **FN1·ALDH2**. (Abstract 표기 OR: FN1=1.32, ALDH2=0.76)

**논문 vs 재현**: 논문 본문은 "LASSO→4모델"인데 **저자 코드는 LASSO∩SVM-RFE 교집합** 사용 — 세부는 다르나 **FN1·ALDH2 결론 동일**.

## F. 임상 상관 (Nephroseq, Figure 5A–B)

**왜**: FN1·ALDH2가 실제 **신장 기능(eGFR)** 과 연결되나.

- 정의: **eGFR** = 추정 사구체여과율(신기능, 높을수록 건강). **r** = 피어슨 상관 −1~+1
- **결과**: ALDH2 vs eGFR **r=+0.74**(보호) / FN1 vs eGFR **r=−0.75**(위험)
- → MR(인과)·ML(진단)에 더해 **임상 지표와도 일치** = 위험/보호 재확인

**논문 vs 재현**: 우리도 Nephroseq 값으로 재현 (`Nephroseq_eGFR_corr.png`), 방향 일치.

## G. 분자 도킹 — 레스베라트롤 (Figure 5C–D)

**왜**: FN1·ALDH2를 **동시에** 조절하는 약 찾기 (진단→치료 연결).

**과정 (정의)**
- **DSigDB**(약물-유전자 DB)에서 FN1↓ 약 79개 + ALDH2↑ 약 35개 → **교집합 5개** → 안전성 종합 **레스베라트롤** 선택
- **분자 도킹**: 약이 단백질에 잘 끼워지나 시뮬레이션. **Vina score**: 더 음수일수록 강함, **<−5면 유의**
- 구조: FN1(PDB 3m7p) / ALDH2(PDB 8dr9)

**결과**: Vina **−7.0(FN1), −9.8(ALDH2)** → 둘 다 강한 결합. **이중표적**(FN1↓·ALDH2↑ 동시) = 시스템약리학. 남들 실험(Qiao·Huang=FN1↓, Su·유도체=ALDH2↑)도 뒷받침. 향후 동물검증 + SGLT2억제제 병용.

**논문 vs 재현**: 도킹은 웹툴(CB-Dock2)이라 **재현 안 함**. 약물 후보 목록(Table S13)은 확인.

## H. scRNA (단일세포) — 어느 세포·언제 (Figure 6~9)

**왜**: bulk는 조직 평균 → "어느 세포에서" 모름. FN1·ALDH2가 **어느 세포·어느 단계**에서 변하나.

**과정 (정의)**: QC(저품질 세포 제거) + **Harmony 배치보정** → 마커로 세포 종류 식별 → **pseudotime**(손상 진행 순서 시간축) → **ssGSEA**(세포별 경로 점수) → **CellChat**(세포간 통신)

**결과 — 초기 vs 말기 대비 (핵심)**

| | 초기 DKD (Fig 6·8) | 말기 DKD (Fig 7·9) |
| --- | --- | --- |
| 세포 | 12종, 손상세관 dPCT1/2 증가 | 신장5+**면역7종**, dPCT 확장 |
| 면역 | 미미 | **대거 침윤** |
| FN1·ALDH2 | **변화 없음** | FN1↑(MES 섬유화)·ALDH2↓ |
| 손상 궤적 | 정상PCT→dPCT1→dPCT2 | 정상PCT→dPCT |
| 통신 | 세관 중심, dPCT 소통↓ | **면역 중심**, dPCT↔면역 MIF·SPP1 |

- **시소 패턴**: 초기엔 ALDH2↓ 바닥에서 **FN1↑ 시작**. 말기엔 ALDH2 잠깐 오르다 꺾임(보호 실패)
- **경로**(Fig 8·9): 초기·말기 공통=대사·재흡수 붕괴 / 말기 고유=**만성염증·EMT**
- **소변 scRNA**(S8): 말기>초기 FN1·ALDH2↑ → 비침습 마커 가능성
- **결론**: FN1·ALDH2는 **말기(진행) 마커**. FN1의 범인 세포 = **MES(메산지움)**, ALDH2 위치 = **근위세관(PCT)**

**논문 vs 재현**: 우리 step7 재현 有 (UMAP·DotPlot·Violin·KEGG.ssGSEA·CellChat). pseudotime(monocle)·일부는 미실행. 소변 GSE266146선 FN1 방향이 조직과 달라 **주의점** 기록.

## I. Discussion 핵심 (중복 제거, 새 포인트만)

- **총괄**: 5가지 방법 통합·**교차검증**이 novelty. 신규 발견이 아닌 **엄밀 검증**.
- **세관 궤적**: 남들은 AKI를 불연속 7아형으로 봤지만, 우리는 **연속 궤적**(정상→dPCT1→dPCT2). 중간 **dPCT1 = 치료 창(개입하면 되돌릴 여지)**. 자멸사·노화 관점 추가.
- **FN1**: 여러 장기 섬유화 공통 = **범섬유화 마커**. 단백뇨 전에도↑(조기). 말기 MES가 섬유화 실행세포.
- **ALDH2**: 알데하이드 해독 **수호효소**(PCT 집중). 결핍→FN1↑·손상 취약. 조직↓인데 소변↑(누출/보상분비 가설) → 액체생검 마커.
- **레스베라트롤**: FN1↓+ALDH2↑ 이중표적. 안전한 건강기능식품이라 신약재창출 유리.
- **맥락**: 정밀의료·비침습진단·단일세포신장학·AI 트렌드에 기여.

## L. 세미나 발표 팁 & 예상 질문

**시간 배분(60분)**: 배경 5 / DEG·경로 8 / **MR 15(핵심)** / 교집합·ML 10 / 임상·도킹 7 / scRNA 10 / 결론·한계 5

**예상 질문 & 답**
- *"왜 MR을 하나?"* → DEG는 상관뿐, 치료표적은 원인이어야 함. SNP(무작위·불변)로 인과 추정.
- *"기울기는 뭘로 계산?"* → beta_DKD÷beta_발현, SNP 약분되어 발현→DKD.
- *"3개 vs 214개?"* → 3=FN1의 SNP 수, 214=유의 유전자 수(5,430개 반복). 층이 다름.
- *"왜 세포단위(scRNA)까지?"* → bulk는 "FN1이 올랐다"까지만, "말기 MES에서 섬유화하며 켜진다"는 기전은 scRNA로만.
- *"재현 결과 차이는?"* → 방향·패턴은 일치, 수치차는 threshold·IV 수·노출 스케일 때문(오류 아님).
