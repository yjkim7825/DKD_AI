## 파일 형태 요약 (다운로드 시)
| 데이터 | 받을 파일 | 비고 |
|---|---|---|
| GEO bulk | `GSExxxxx_series_matrix.txt.gz` + 해당 `GPLxxxx` 주석 | probe→gene 필요 |
| GEO scRNA | `*_RAW.tar` / 10x (`matrix/barcodes/features`) | Seurat/scanpy 입력 |
| FinnGen | `finngen_R12_DM_NEPHROPATHY_EXMORE.gz` | MR outcome |
| GWAS Catalog | `GCST90435706` harmonised TSV | MR outcome |
| eQTL(FN1/ALDH2) | OpenGWAS API / eQTLGen cis-eQTL | MR exposure |
| gene set | MSigDB Hallmark(.gmt), KEGG(.gmt) | GSEA/ssGSEA |
| figshare | `Article related data.zip` (4.3MB) | 저자 코드·결과 |

> **그룹 라벨 규약**: 발현행렬 샘플 열 = `{샘플ID}_{그룹}` (예 `GSM123_Control`, `GSM456_DKD`).
> Early/Late 통합 2군 비교 시 그룹을 `DKD`로.
