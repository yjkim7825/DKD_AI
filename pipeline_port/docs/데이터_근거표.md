## 2. GWAS / eQTL (MR 노출)
- 논문 근거: Methods > *Data collection* (p.3) — "exposure GWAS ... from the **OpenGWAS** database
  (https://gwas.mrcieu.ac.uk/, accessed Jan 15 2025) ... genome-wide significant **eQTL** SNPs
  (p<5×10⁻⁸), LD clumping (r²<0.001/10,000kb) → 26,152 SNPs / 5,430 genes."
- ENSG ↔ 유전자: **FN1 = ENSG00000115414, ALDH2 = ENSG00000111275** (표준 Ensembl).
  받은 `eqtl-a-ENSG…` = OpenGWAS **eqtl-a 배치(= eQTLGen cis-eQTL)**. 이 둘은 논문 최종 후보(FN1·ALDH2) →
  **최종 MR 검증 대상 노출**.
- 코드 근거: 원본 `Mendelian randomization 1.R` (TwoSampleMR `read_exposure_data`).

## 3. DKD outcome GWAS
- 논문 근거: Methods (p.3) — "**DM_NEPHROPATHY_EXMOR** dataset (64,663) from FinnGen R12" +
  "**GCST90435706** (388,955) from GWAS Catalog ... as outcome GWAS for DKD".
  MR 방법: p.4 *Mendelian randomization* (TwoSampleMR v0.6.16, IVW primary).
- Data availability (p.21): FinnGen `…finngen_R12_DM_NEPHROPATHY_EXMORE.gz`, GCST `…/GCST90435706/`.
- 코드/파일 근거: 원본 `Mendelian randomization 1.R`(outcome=finngen…gz),
  figshare **Supp Table 6**(FinnGen MR) / **Supp Table 7**(GWAS Catalog MR).
- ⚠️ 표기 주의: 논문 본문 `…EXMOR`(잘림) ↔ 실제 파일명 `…EXMORE`(끝 E).
  GCST90435706을 Methods는 "DKD outcome", Data availability(p.21)는 "**CKDGen eGFR GWAS**"로 적어
  **논문 자체 표기가 약간 불일치** — 어쨌든 두 번째 outcome GWAS.

## 4. Pathway / Gene set
- 논문 근거: Methods (p.3) — "Hallmark and **KEGG** pathways ... from **MSigDB** [ref 12]".
  ssGSEA는 "**186 KEGG gene sets** curated from MSigDB" (p.273). 전체 목록은 Table S2.
- 코드 근거: figshare `hallmark.gsea.R`(line 18 `msigdbr(category="H")` = Hallmark),
  원본 `GSEA.R`(`c2.cp.kegg.Hs.symbols.gmt` = KEGG).
  → 받은 `4-1 h.all…gmt`(Hallmark 50), `4-2 c2.cp.kegg_legacy…gmt`(≈186)가 이에 대응.

## 5. Nephroseq (FN1·ALDH2 ↔ eGFR)
- 논문 근거: Methods (p.~4) — "**Nephroseq v5** database (http://v5.nephroseq.org, accessed Mar 4 2025)
  ... relationship with **estimated GFR**". Results — "correlations of **ALDH2 and FN1 with eGFR**".
- ⚠️ 받은 파일은 **FN1만**(5-1 disease, 5-2 GFR corr). **ALDH2 Nephroseq 없음** — 필요 시 동일 쿼리 추가.
