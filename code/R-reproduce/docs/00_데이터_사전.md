# DKD 데이터 사전 & 사용 가이드

멘델리안 무작위화(MR) 중심 파이프라인. **노출(원인)=eQTL → 결과=DKD GWAS**,
여기에 발현·경로·임상 데이터가 보조로 붙는다.

---

## 2. GEO 발현 데이터 — DEG·머신러닝·single-cell 분석용 : 8개

- GEO (Gene Expression Omnibus) = 미국 NCBI가 운영하는 세계 최대 유전자 발현 데이터 공개 저장소
  - 일반적으로 bulk가 환자 수가 많아 통계 비교에 유리하기 때문에 사용한 것으로 추정
- https://www.ncbi.nlm.nih.gov/geo/
  - 데이터 설명

### 1. bulk RNA-seq : 5개

- **1-1 GSE96804 : 학습(training) — DEG·LASSO 학습셋**
  - 플랫폼: 마이크로어레이 (HTA-2.0)
  - 조직: 사구체 (필터부)
  - 구성: DKD 환자 + 정상
  - 역할: 학습셋 — 진단 모델을 여기서 훈련
  - 행 = 유전자 70,523개
  - 열 = 환자 61명
- **1-2 GSE104948 : 독립 검증(validation) — ROC 확인**
  - 플랫폼: 마이크로어레이 2종(U133A + U133Plus2) → 파일 2개
  - 조직: 사구체
  - 구성: 여러 신장질환 혼재(DKD 포함)
  - 역할: 독립 검증셋 — 학습 안 한 데이터로 성능 확인
  - 행 = 프로브(유전자): U133A는 22,283개 / U133Plus2는 54,675개
  - 열 = 환자 196명 (U133A 125명 + Plus2 71명)
- **1-3 GSE104954 : 독립 검증(validation)**
  - 플랫폼: 마이크로어레이 2종
  - 조직: 세뇨관 간질 (재흡수부)
  - 역할: 독립 검증셋
  - 행 = 프로브(유전자): 22,283개 / 54,675개
  - 열 = 환자 195명 (135명 + 60명)
- **1-4 GSE142025 : 초기 vs 말기 비교**
  - 플랫폼: RNA-seq (그래서 유전자가 심볼로 나옴)
  - 조직: 신장
  - 구성: 정상(N) / 초기(B) / 말기(A) — 3단계
  - 역할: 초기 vs 말기 비교의 핵심
  - 행 = 유전자 17,184개 (RNA-seq라 심볼로 나옴)
  - 열 = 환자 36명 (정상 N + 초기 B + 말기 A)
- **1-5 GSE30529 : 단계 비교 재검증**
  - 플랫폼: 마이크로어레이 (U133A_2)
  - 조직: 세뇨관 간질
  - 구성: DKD + 정상
  - 역할: 단계 비교 결과 재검증
  - 행 = 프로브(유전자) 22,277개
  - 열 = 환자 22명 (DKD + 정상)

### 2. single-cell RNA-seq : 3개

- **1-6 GSE131882 : 신장 조직 single-cell**
  - 플랫폼: single-cell RNA-seq (zUMIs 10x)
  - 조직: 신장 조직
  - 구성: 정상 3샘플 + 당뇨 3샘플 = 6개
  - 행 = 세포 3,904개 (바코드 AAACCTGAGTGTTAGA 같은 ID)
  - 열 = 유전자 36,773개 — ENSG(Ensembl ID)로 나옴 (심볼 아님)
  - 특이점: 유전자가 이름(FN1) 아니라 ID(ENSG00000115414)라, 찾을 때 ID로 변환 필요
- **1-7 GSE209781 : 신장 조직 single-cell**
  - 플랫폼: single-cell RNA-seq (10x Cell Ranger)
  - 조직: 신장 조직
  - 구성: 정상(NM) + DKD
  - 행 = 세포: 원시 679만 → QC 후 24,104개 (원시엔 빈 방울이 대부분, 걸러내면 진짜 세포만)
  - 열 = 유전자 23,934개 (QC 후, 심볼로 나옴)
  - 특이점: 세포가 많아 무겁고, QC(품질 필터)가 꼭 필요
- **1-8 GSE266146 : 소변 침전물(urinary sediment) single-cell**
  - 플랫폼: single-cell RNA-seq (BD Rhapsody)
  - 조직: 소변 침전물 ← 신장 조직을 안 떼고 소변으로 (비침습적)
  - 구성: 초기(Early) + 말기(Adv) SampleTag로 구분
  - 행 = 세포 162개
  - 열 = 유전자 22,973개
  - 특이점: 소변이라 세포 수가 적음(162개)

## 3. eQTL 데이터 — MR 노출(원인 쪽) : "유전자 발현을 조절하는 SNP"로 발현을 대리

- eQTL SNP 선별(p<5×10⁻⁸) → LD 클럼핑(10,000kb 내 r²<0.001) → 5,430개 유전자의 독립 SNP 26,152개 확보
  - → 약하거나 중복된 SNP를 쓰면 MR이 편향되니 엄격히 걸러 깨끗한 도구변수만
- https://opengwas.io/
  - 데이터 설명
- **2-1 FN1 eQTL**
  - SNP × 통계, 18,851행 10열
  - 플랫폼: VCF (GWAS-VCF 형식)
  - 무슨 데이터: FN1 발현을 조절하는 SNP 목록
  - 행 = SNP 18,851개 / 열 = 위치·대립유전자·효과크기(beta)·오차(SE) 등 10개
  - 역할: MR의 원인 쪽(FN1 노출)
- **2-2 ALDH2 eQTL**
  - SNP × 통계, 15,636행 10열
  - 무슨 데이터: ALDH2 발현 조절 SNP
  - 행 = SNP 15,636개 / 열 = 10개
  - 역할: MR의 원인 쪽(ALDH2 노출)
- **2-3 eQTLGen cis-eQTL (원본)**
  - SNP × 통계, 수천만행 15열 (`.txt.gz` 3.7GB)
  - 플랫폼: TSV(gzip)
  - 무슨 데이터: eQTLGen 전체 cis-eQTL — 2-1/2-2의 원재료(대체·검증 경로)
  - 행 = SNP–유전자 쌍 / 열 = Pvalue, SNP, 위치, AssessedAllele, OtherAllele, **Zscore**, Gene, NrSamples 등 15개
  - 특이점: beta·SE 없음(**Zscore만**) → 2-4 빈도로 beta/SE 역산해야 MR 가능
  - 역할: 노출을 원본에서 직접 만들 때 / 통째 로드 금지, 청크 스트리밍 + Gene 필터
- **2-4 allele frequency**
  - SNP × 빈도, 10,823,015행 9열
  - 무슨 데이터: 대립유전자 빈도표
  - 역할: F통계량 계산·harmonization(방향 맞추기)

## 4. 결과 GWAS 데이터 — MR 결과(DKD 발병 쪽)

- 인종이 섞이면 병 때문인지 인종 차이 때문인지 헷갈리니 → 두 데이터 모두 한 인종(유럽계)으로 통일
  - → 두 집단을 쓴 이유: 한 곳에서만 나온 우연이 아닌지 재확인
- **3-1 FinnGen R12 — DM_NEPHROPATHY_EXMORE (핀란드, 64,663명)**
  - https://www.finngen.fi/en/access_results
  - SNP × 통계, 21,191,346행 13열 (2.2GB)
  - 플랫폼: TSV
  - 무슨 데이터: 핀란드인의 DKD 유전연관 (SNP↔DKD)
  - 행 = SNP 2천만+ / 열 = beta·SE·p값 등 13개
  - 역할: MR의 결과 쪽(핀란드)
- **3-2 GWAS Catalog — GCST90435706 (영국, 388,955명)**
  - https://www.ebi.ac.uk/gwas/
  - SNP × 통계, 28,159,587행 17열
  - 무슨 데이터: 영국인의 DKD 유전연관
  - 행 = SNP 2천8백만+ / 열 = 17개
  - 역할: MR의 결과 쪽(영국)

## 5. 경로 데이터 — 기능(경로) 분석용 (GSEA/ssGSEA의 "정답지" = Table S2)

- MSigDB (Hallmark·KEGG) — 염증반응·EMT·세포자멸사
  - https://www.gsea-msigdb.org/gsea/msigdb/collections.jsp
    - **4-1 Hallmark**
      - 경로 × 유전자명단, 50행 (열 가변)
      - 플랫폼: GMT (직사각형 표 아님)
      - 무슨 데이터: 대표 병리경로 50종에 속한 유전자 명단
      - 행 = 경로 50개 / 각 줄에 [경로명, URL, 소속 유전자들...]
      - 역할: 경로 강화분석의 "정답지"
    - **4-2 KEGG**
      - 경로 × 유전자명단, 186행 (열 가변)
      - 무슨 데이터: KEGG 경로 186종의 유전자 명단
      - 역할: 경로 강화분석의 정답지
- Human Autophagy Database — 자가포식
  - http://www.autophagy.lu/
  - 저자 제공 데이터인 Table S2에 반영됨
- 발표 논문들(문헌) — 세포노화·산화스트레스
  - 저자 제공 데이터인 Table S2에 반영됨
- **▷ Table S2 내용 — 경로별 유전자 목록 (GSEA의 "정답지")**
  - 경로 × 유전자, 약 1,400개 유전자 (열 = 경로, 각 열 아래로 소속 유전자 심볼 나열)
  - 6개 경로: 염증반응(Inflammatory response) / 상피-간엽 전환(EMT) / 세포자멸사(Apoptosis) / 세포노화(Cellular senescence) / 자가포식(Autophagy) / 산화스트레스(Oxidative stress)
  - 예: 염증반응 = ABCA1, ABI1, ACVR1B... / EMT = ABI3BP, ACTA2, ADAM12...
  - 위 Human Autophagy DB(자가포식)·문헌(세포노화·산화스트레스)이 이 표의 해당 열로 합쳐져 반영됨
  - 역할: GSEA/ssGSEA에서 "이 경로에 이 유전자들이 속한다"는 기준표(정답지)

## 6. 임상·치료 데이터

- Nephroseq — FN1·ALDH2와 eGFR(신기능) 상관분석 (파일 5-1~5-4)
  - https://nephroseq.org/
    - **5-1 FN1 Disease**
      - 분석결과 × 통계, 5행
      - 무슨 데이터: FN1 발현 질병군 vs 정상 결과
      - 행 = 분석 5건 / 열 = p값·Fold Change·r값 등
      - 역할: 임상 검증
    - **5-2 FN1 GFR**
      - 분석결과, 4행 — FN1 vs eGFR 상관
    - **5-3 ALDH2 Disease**
      - 분석결과, 0행 — 유의한 결과 없음
    - **5-4 ALDH2 GFR**
      - 분석결과, 1행 — ALDH2 vs eGFR 상관
      - 특이점: 원시 데이터가 아니라 이미 요약된 분석 결과라 행이 적음
- DSigDB — FN1·ALDH2 표적 치료제 후보 스크리닝 (결과 = Table S13)
  - http://dsigdb.tanlab.org/DSigDBv1.0/
  - 저자 제공 데이터인 Table S13에 반영됨
  - **▷ Table S13 내용 — 치료제 후보 (신약재창출)**
    - 약물 × 통계, 114개 화합물 4열 (FN1 낮추는 약 79개 + ALDH2 높이는 약 35개)
    - 열 4개: Drug(약물명) / Gene(표적=FN1 또는 ALDH2) / Type(효과 방향+세포주, 예 `0.0000122=DOWN(HL60)`) / Source(출처, 예 CMAP)
    - 예: ajmaline → FN1 DOWN, alexidine → FN1 DOWN...
    - 역할: FN1↓·ALDH2↑ 효과를 내는 기존 약물 후보 발굴(MR 결론 기반 신약재창출)
