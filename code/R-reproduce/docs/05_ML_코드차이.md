## 1. 저자 코드 재현 — `reference/ml_author.R` (= ML1/2.R)

**원본 직접 대조 완료** (업로드된 `machine learning modeling 1.R`·`2.R` ↔ `ml_author.R`): 네 단계 모두 로직 일치. ML2의 "유전자별 학습·검증 ROC(95%CI)"도 `ml_author.R` B4에서 동일하게 학습+검증 둘 다 생성하도록 반영함.

### ① LASSO — ★ 동일 (저자 ML1.R)
**저자**
```r
set.seed(123); library(glmnet)
fit=glmnet(x, y, family="binomial", alpha=1)
cvfit=cv.glmnet(x, y, family="binomial", alpha=1, type.measure='deviance', nfolds=10)
coef=coef(fit, s=cvfit$lambda.min); lassoGene=row.names(coef)[which(coef!=0)][-1]
```
**우리** — 동일 (`cv.glmnet` + `lambda.min` + 계수≠0). 글자 단위로 같음.

### ② SVM-RFE — ★ 동일 (저자 ML1.R)
**저자**
```r
library(e1071); library(kernlab); library(caret)
Profile=rfe(x=rt, y=as.numeric(group), sizes=c(2:8),
            rfeControl=rfeControl(functions=caretFuncs, method="cv"),
            methods="svmRadial", preProcess=c("center","scale"))
featureGenes=Profile$optVariables
```
**우리** — 동일 (`caret::rfe` + `svmRadial` + sizes 2~8).

### ③ 교집합(Venn) — ★ 동일 (저자 ML1.R)
```r
geneList=list(LASSO=..., "SVM-RFE"=...); intersectGenes=Reduce(intersect, geneList)
```
저자·우리 모두 `venn.diagram` + `Reduce(intersect)`.

### ④ 유전자별 ROC — ★ 동일 (저자 ML2.R)
```r

```
학습(`data.train`)·검증(`data.test`) 둘 다 유전자별 ROC. 우리도 동일.

→ **저자 코드는 로직 그대로 재현됨.** 다만 실행하면 **VNN2 1개로 붕괴**(FN1·ALDH2 못 잡음) → 본문 결과와 안 맞음.

## 2. 논문 본문 방식 — `01_ml_feature.R` + `02_ml_model.R` (채택, 저자 코드엔 없는 부분)

저자 업로드 코드엔 **빠져 있지만 논문 본문·Figure 4엔 있는** 것들을 구현:

### ⓪ 배치보정 (저자 코드 ✗ / 논문 ✓)
```r
combat <- sva::ComBat(dat=comb, batch=배치(학습/검증), mod=NULL, par.prior=TRUE)
```
- 논문 Methods "데이터셋 간 배치효과 sva로 보정" + Fig 4A(보정 후 PCA) → 우리 `A_PCA_batch.pdf`.

### ① ROC-AUC>0.8 필터 (저자=교집합 / 논문=ROC필터)
```r
for (g in lassoGene) if (auc_tr>0.8 & auc_te>0.8) final <- c(final, g)   # 학습·검증 둘 다
```
- 논문: LASSO 6개 → **학습·검증 모두 AUC>0.8** 인 유전자만 다변량 모델로 → FN1·ALDH2.
- **저자 SVM-RFE 교집합보다 이게 더 안정적**(검증에서 무너지는 유전자를 명시적으로 제거).

### ② 4모델 결합 (저자 ✗ / 논문 ✓ Fig4H·I)
```r
glm(); randomForest(); svm(kernel="radial"); xgb.train(objective="binary:logistic")
# 각 모델 학습·검증 ROC 곡선 → A_Figure4_DI.pdf (H·I 패널)
```
- 논문: FN1+ALDH2를 4모델로 결합 → GLM 최고(검증 0.942). 우리 재현 GLM 0.972.

### ③ Figure 4 D~I 그림 (저자 ✗ / 우리 ✓)
- `A_Figure4_DI.pdf`: D~G 유전자별 ROC(학습·검증) + H·I 4모델 ROC 곡선(AUC+95%CI).
