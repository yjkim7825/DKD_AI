## 2. 변화 구간 3분류 (정상↔초기↔말기)

DEG가 **언제 변하나**를 3구간 조합으로 분류:

| 구간 | 정의 | 개수 |
| --- | --- | --- |
| ① 초기에만 | 정상→초기(N→E)만 변함 | **432** |
| ② 둘 다 (계속) | N→E ∩ 초기→말기(E→L) | **239** |
| ③ 말기에만 | E→L만 변함 | **3,075** |

→ **대부분(3,075)이 말기에 새로 터짐**, 초기부터 계속 변하는 건 239개뿐.

## ※ 통계 검정
- DEG 유의성은 **limma `eBayes`(조정 t통계량)** + **Benjamini-Hochberg FDR**(adj.p<0.05, |logFC|>0.585). 논문 통계 파트의 "t-검정"에 대응하나 엄밀히는 **limma 조정-t**(단순 t검정 아님). 전체 통계: `00_통계방법.md`
